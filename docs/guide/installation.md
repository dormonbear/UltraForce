# Installation

## Chrome Web Store

Install UltraForce from the Chrome Web Store:

1. Visit the [UltraForce Chrome Web Store page](https://chromewebstore.google.com/detail/ultraforce-for-salesforce/maemkmihjmlfilhpfeeindecjnagelkh)
2. Click "Add to Chrome"
3. Confirm the installation

## Permissions

| Permission | Purpose |
|------------|---------|
| `storage` | Store cached metadata and settings |
| `activeTab` | Access the current Salesforce tab |
| `tabs` | Open new tabs for navigation |
| `cookies` | Read Salesforce session for API access |

## Verify Installation

1. Navigate to any Salesforce org
2. Press `Cmd+B` (Mac) or `Ctrl+B` (Windows)
3. The UltraForce search modal should appear
