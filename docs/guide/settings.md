# Settings

Open settings by clicking the gear icon in the search modal.

## Search Settings

### Fuzzy Search

Enable or disable fuzzy matching. When enabled, searches like `acnt` will match `Account`.

### Search Scope

Configure which metadata types to include in searches.

## Display Settings

### Theme

UltraForce uses a dark transparent theme that adapts to your Salesforce interface.

### Result Limit

Configure the maximum number of search results displayed.

## Cache Settings

### Cache Duration

Metadata is cached for 24 hours with background refresh after 2 hours.

### Clear Cache

Manually clear the metadata cache to force a fresh fetch.

## API Statistics

View your API usage:

- **Last 24 hours** - Requests in the past day
- **Last 30 days** - Requests in the past month
- **Total** - All-time request count

## Keyboard Shortcuts

Customize the keyboard shortcut to open UltraForce in Chrome's extension settings:

1. Go to `chrome://extensions/shortcuts`
2. Find UltraForce
3. Set your preferred shortcut
