# UltraForce UI Design Guide

## Overview

UltraForce的搜索界面采用Raycast风格设计，追求简洁、现代、优雅的用户体验。

## 核心设计原则

### 1. 透明毛玻璃效果 (Glassmorphism)
- 主背景：`rgba(25, 30, 35, 0.45)` - 透明的暗色背景
- 毛玻璃效果：`blur(80px) saturate(180%) brightness(0.65)` - 强模糊，高饱和度，适中亮度
- 边框：`1px solid rgba(255, 255, 255, 0.08)` - 微妙的白色边框
- 圆角：`20px` - 大圆角营造现代感

### 2. 视觉层次
- **主容器**：透明深色背景，强烈毛玻璃效果
- **搜索区域**：更透明的背景 `rgba(255, 255, 255, 0.01)`，与下方结果区域形成分界
- **结果项**：hover时背景 `rgba(255, 255, 255, 0.04)`，选中时蓝色背景

### 3. 色彩规范

#### 背景色
- 主容器背景：`rgba(25, 30, 35, 0.45)`
- 搜索输入区：`rgba(255, 255, 255, 0.01)`
- 类型过滤器：`rgba(255, 255, 255, 0.03)`
- 结果分组标题：`rgba(255, 255, 255, 0.01)`

#### 文字色
- 主要文字：`rgba(255, 255, 255, 0.95)` - 高对比度白色
- 次要文字：`rgba(255, 255, 255, 0.7)` - 中等透明度白色
- 辅助文字：`rgba(255, 255, 255, 0.5)` - 低透明度白色
- 占位符：`rgba(255, 255, 255, 0.4)` - 更低透明度白色

#### 交互态
- hover：`rgba(255, 255, 255, 0.04-0.08)`
- selected：`rgba(59, 130, 246, 0.15)` - 蓝色半透明
- active：`rgba(59, 130, 246, 0.25-0.35)` - 更深的蓝色

### 4. 布局特点

#### 一体化设计
- 搜索框与结果列表无分割线，形成一体化容器
- 移除所有不必要的视觉元素（如搜索图标）
- 紧凑的间距，最大化内容显示区域

#### 间距规范
- 容器内边距：`16px 20px`
- 结果项内边距：`8px 20px`
- 图标与文字间距：`6px`
- 组件间间距：`8px`

#### 圆角规范
- 主容器：`20px`
- 小组件（按钮、标签）：`6px-8px`
- 输入框：`12px`

### 5. 字体规范

#### 字体族
```css
font-family: -apple-system, BlinkMacSystemFont, 'SF Pro Display', 'Segoe UI', sans-serif;
```

#### 字体大小
- 搜索输入框：`16px`
- 结果项主标题：`14px`
- 结果项副标题：`12px`
- 分组标题：`11px`
- 快捷键提示：`11px`

#### 字重
- 常规文字：`400` (Regular)
- 标题文字：`500` (Medium)
- 分组标题：`600` (Semibold)

### 6. 交互效果

#### 毛玻璃渲染
```css
backdrop-filter: blur(80px) saturate(180%) brightness(0.65);
-webkit-backdrop-filter: blur(80px) saturate(180%) brightness(0.65);
```

#### 阴影层次
```css
box-shadow: 
  0 25px 80px rgba(0, 0, 0, 0.8),    /* 主阴影 */
  0 12px 40px rgba(0, 0, 0, 0.6),    /* 中层阴影 */
  inset 0 1px 0 rgba(255, 255, 255, 0.05),    /* 内部高光 */
  inset 0 -1px 0 rgba(0, 0, 0, 0.3); /* 内部阴影 */
```

#### 过渡动画
```css
transition: all 0.2s ease;  /* 标准过渡 */
transition: all 0.15s ease; /* 快速过渡 */
transition: all 0.1s ease;  /* 极速过渡 */
```

### 7. 响应式设计

#### 容器尺寸
- 宽度：`90%`，最大 `680px`
- 高度：最大 `70vh`
- 位置：垂直居中偏上 `top: 15vh`

#### 移动端适配
- 在小屏幕设备上保持比例缩放
- 触摸友好的按钮尺寸
- 适当的间距调整

### 8. 状态设计

#### 空状态
- 友好的提示文案
- 清晰的操作指引
- 保持一致的视觉风格

#### 加载状态
- 微妙的加载指示器
- 不破坏整体布局
- 快速响应用户操作

#### 错误状态
- 明确的错误提示
- 建设性的解决方案
- 保持用户友好的语调

## 实现要点

### CSS关键样式
1. **透明背景**：使用rgba值控制透明度
2. **毛玻璃效果**：backdrop-filter实现模糊效果
3. **边框**：微妙的白色边框增强层次
4. **阴影**：多层阴影创造深度感
5. **过渡**：流畅的状态切换动画

### 设计一致性
1. **色彩统一**：所有透明度级别保持一致逻辑
2. **间距规整**：使用8px基准网格系统
3. **圆角协调**：大小组件圆角成比例
4. **字体层次**：清晰的信息层级

### 性能考虑
1. **GPU加速**：利用transform和opacity触发硬件加速
2. **重绘优化**：避免频繁的布局变化
3. **资源优化**：合理使用模糊效果，避免性能问题
